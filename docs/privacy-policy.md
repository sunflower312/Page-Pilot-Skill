# Privacy Policy

Page Pilot Skill 主要在本地运行。

## Collected Data

- 本地 benchmark 和验证过程中产生的截图、DOM 快照、JSON 报告
- 用户显式打开页面后返回的结构化页面信息

## Storage

- 这些数据默认写入本地 `artifacts/page-pilot-skill/`
- 仓库默认通过 `.gitignore` 忽略这些运行产物

## Network Access

- Page Pilot Skill 会按用户要求访问目标网页
- benchmark 会访问公开练习 / 演示站点

## Responsibility

请不要在未经授权的网站上使用本工具进行自动化测试或数据抓取。
